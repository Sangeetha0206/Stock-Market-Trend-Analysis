# Stock Market Trend Analysis

This project analyses stock market trends using Python.  
It includes:
- Stock Market Data CSV
- Analysis Notebook with visualizations
- Key insights from trends

## How to Run
1. Open `Stock_Market_Trend_Analysis.ipynb` in Google Colab.
2. Run all cells.
3. CSV data (`stock_data.csv`) is used for analysis.

## Output
- Charts and trend analysis displayed in notebook.
